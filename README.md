# TSPTW-C Brazilian Logistics

Este repositório contém a implementação do modelo TSPTW-C (Travelling Salesman Problem with Time Windows and Constraints), uma extensão do Problema do Caixeiro Viajante com Janelas de Tempo aplicada ao contexto logístico brasileiro.

O projeto integra:

- Restrição de janelas de tempo dos clientes;
- Capacidade máxima de carga do veículo;
- Consumo de combustível dependente do peso transportado;
- Custos operacionais (combustível, manutenção, mão de obra e esperas);
- Controle de jornada do motorista conforme legislação brasileira;
- Heurística construtiva para geração de rotas.

A implementação foi desenvolvida como parte de um estudo acadêmico aplicado ao roteamento de distribuição entre municípios do Rio Grande do Norte.
